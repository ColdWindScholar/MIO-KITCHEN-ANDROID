# Быстрый старт

MIO-KITCHEN — это однопакетная firmware-kitchen для Android. Она поддерживает
распаковку, модификацию, пересборку и прошивку прошивок для устройств
Android 10 — Android 15+ — всё из одного приложения, без отдельных "legacy"
и "modern" сборок.

## Установка

1. Установите `MIO-KITCHEN-<date>-<version>.apk` на любое устройство
   Android 5.0+ (API 21+).
2. При первом запуске выдайте runtime-разрешения, которые запросит приложение:
   - На Android 13+: **Уведомления** (чтобы foreground-сервис мог показывать
     прогресс).
   - На Android ≤ 12: **Доступ к хранилищу** (legacy-разрешение, ограничено
     API 32).

Root **опционален** — большинство операций распаковки/упаковки работают без
root. Root требуется только для flash-операций и для чтения файлов вне
app-specific workspace.

## Выбор прошивки

1. Нажмите **Pick firmware file** на главном экране.
2. Откроется системный выбор файлов (SAF). Выберите `.zip` OTA, `.img`-файл
   или любой firmware-пакет.
3. Приложение копирует файл в workspace, считает SHA-256 и запускает
   `FirmwareAnalyzer`, чтобы определить тип пакета, capabilities и подсказку
   версии Android.
4. Найденный профиль показывается на экране: `type=payload_bin, android=13,
   dynamic, erofs, avb`.

## Запуск операции

1. После анализа выберите операцию из списка доступных:
   - **Unpack ROM** — извлечь всё из OTA zip.
   - **Unpack boot.img** — разделить boot на kernel + ramdisk.
   - **Unpack super.img** — разделить dynamic partitions.
   - **Pack** — пересобрать изменённый образ.
   - **Verify vbmeta** — проверить AVB-цепочку.
   - **Flash prepare** — подготовить flash-скрипт (деструктивно, требует root).
2. Приложение строит `OperationPlan` и показывает блокировки (отсутствующие
   инструменты, нет root для flash и т.д.). Устраните блокировки перед
   продолжением.
3. Нажмите **Execute**. Появится foreground-notification
   `FirmwareOperationService`; `OperationExecutor` запускает операцию через
   `ShellRuntime`.

## Экспорт результата

1. После успешной операции результат лежит в workspace приложения.
2. Нажмите **Export** и выберите, куда сохранить:
   - Выберите папку один раз (TreeFolder) для массового экспорта, или
   - Выбирайте назначение для каждого файла (AskPerFile).
3. SHA-256 считается во время экспорта, чтобы вы могли проверить результат.
